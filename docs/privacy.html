Privacy Policy

The Podku mobile application does not collect any personal data from users. Data is linked to the server they choose to use, whether it is a self‑hosted or third‑party server. The developers of Podku do not operate public servers except for an optional demo server.

When using a Podku server, the following information may be collected:

  • Podcasts subscriptions: What podcasts the user has been subscribing to
  • Podcast episode listening data: when listening to a podcast, the progress will be sent to the user's podku server

The Podku developers neither own nor share the data listed above. All collected data is transmitted exclusively to the host of the server and is never processed for analytics purposes.
